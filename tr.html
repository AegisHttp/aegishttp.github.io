🛡️ Sıfır-Güven Mimarisi

Standart REST API'ler için Mutlak Uçtan Uca Şifreleme

Aegis; veri sızıntılarını, XSS saldırılarını ve üçüncü parti kütüphane tedarik zinciri ihlallerini tamamen engellemek için tasarlanmış sıfır-güven (zero-trust) tabanlı bir uçtan uca şifreleme geçididir. Standart REST isteklerini tarayıcı DOM katmanında yakalayıp, şifreleme işlemlerini tarayıcı belleğinin dışındaki izole bir işletim sistemi seviyesi GPG servisine (daemon) devrederek gizli anahtarların ve ham verilerin çalınmasını imkansız kılar. Hassas API'lerin korunması, işletim sistemi seviyesinde Pinentry doğrulama döngülerinin çalıştırılması ve WKD (Web Key Directory) üzerinden kimlik doğrulaması gibi tüm bu kabiliyetleri, mevcut kod yapınızda hiçbir değişiklik gerektirmeyen tak-çalıştır bir yapıyla sunar.

Çözülen Güvenlik Açıkları

Aegis, modern web mimarilerini tehdit eden en kritik saldırı vektörlerine karşı matematiksel ve mimari koruma sağlar.

XSS (Cross-Site Scripting) ile Anahtar Hırsızlığı

Standart web cüzdanları veya şifreleme çözümlerinde gizli anahtarlar localStorage veya JS Heap bellek alanlarında tutulur. Sayfaya sızan bir XSS atağı bu anahtarları saniyeler içinde çalabilir. Aegis ise anahtarları işletim sistemi seviyesindeki GPG anahtarlığına gömerek zararlı betiklerden tamamen izole eder.

MitM (Ortadaki Adam) & TLS Kırılması

Kurumsal ağ geçitleri veya saldırganlar SSL/TLS sertifikalarını taklit etse ya da HTTPS katmanını tamamen deşifre etse dahi, Aegis üzerinden taşınan paketler zırhlı AES-256 PGP blokları olarak hareket eder. Veriyi sadece hedef sunucu çözebilir.

Tekrar Oynatma Saldırıları (Replay)

Sunucuya gönderilen her şifreli paket, ISO8601 zaman damgası ve benzersiz bir UUID nonce (`_gpg_nonce`) ile mühürlenir. Backend bu nonceları kısa süreli bir TTL cache yapısında denetleyerek mükerrer istekleri anında reddeder.

DOM Decryption Oracle İstismarı

Sayfaya sızan kötü niyetli XSS betiklerinin, eklentiyi şifre çözücü (oracle) olarak kullanarak hassas verileri gizlice açmasını önlemek amacıyla; Aegis, sadece fiziksel onay ekranı (pinentry) ile üretilen kısa ömürlü, hafıza sınırlandırılmış `x-gpg-session-token` jetonları kullanır.

Tüm HTTP Header ve JWT Koruması

Sadece veri gövdesini şifreleyen geleneksel uygulama düzeyindeki şifrelemelerin aksine Aegis, HTTP istek zarfının tamamını (örneğin Authorization: Bearer JWT gibi özel başlıkları) şifreler. Bu sayede oturum jetonlarınız, kurumsal proxy incelemelerinden ve pasif ağ dinlemelerinden mutlak şekilde korunur.

Zararlı Eklenti ve Arka Plan Tarama Direnci

Kullanıcının tarayıcısına sızan zararlı bir eklenti veya bilgisayara yerleşen bir Trojan, kullanıcının açık oturumunu (session cookies) kullanarak arka planda istekleri dinlemeye veya taklit etmeye çalışsa bile başarılı olamaz. Hatta Developer Tools (Geliştirici Araçları) üzerinden Network (Ağ) sekmesi izlense dahi, aradaki tüm HTTP istek ve yanıtları yalnızca GPG şifreli (şifreli metin) olarak görülür. Düz metin verisi tarayıcı dışındaki güvenli bölge haricinde asla ele geçirilemez.

TypeScript İstemci Kurulumu

Ağ isteklerini otomatik yakalayan ve oturum döngüsünü yöneten proxy yapısını başlatın.

aegis-app.ts
import { AegisClient } from 'aegis-ts-sdk';

// Aegis istemci proxy kancasını kurun
const aegis = new AegisClient({
  serverPublicKeyUrl: 'https://keyserver.ubuntu.com/pks/lookup?op=get&search=0xServerKeyID',
  forceTunneling: true // GET/DELETE metot ve başlıklarını şifreli POST gövdesine hapseder
});

// OS anahtarlığındaki GPG imzasıyla otonom kimlik doğrulaması gerçekleştirin
await aegis.login();

// Standart fetch çağrılarınızı yazın - veriler arka planda şeffafça şifrelenir!
const response = await fetch('/api/secure-profile', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    SSN: "000-12-3456",
    balance: 92834.12
  })
});

const data = await response.json();
console.log('Sunucudan gelen deşifre edilmiş yanıt:', data);

GoFiber Middleware Yapılandırması

GPG şifre çözme filtresini uygulamanıza entegre edin. Standart Go controller'larınız veriyi sade JSON olarak işler.

main.go
package main

import (
	"github.com/AegisHttp/gofiber-aegis-http"
	"github.com/gofiber/fiber/v2"
)

func main() {
	app := fiber.New()

	// Aegis Şifre Çözme Middleware'ini global olarak enjekte edin
	app.Use(aegis.New(aegis.Config{
		PrivateKeyRingPath: "/home/server/.gnupg/secring.gpg",
		PrivateKeyPassphrase: "SuperSecretBackendPassphrase",
		KeyserverPublish: true, // Sunucu public anahtarını otomatik olarak Ubuntu sunucularına yükler
	}))

	// Standart yönlendirmeleriniz çözülmüş JSON içeriğiyle çalışmaya devam eder
	app.Post("/api/secure-profile", func(c *fiber.Ctx) error {
		type RequestData struct {
			SSN     string  `json:"SSN"`
			Balance float64 `json:"balance"`
		}

		var req RequestData
		if err := c.BodyParser(&req); err != nil {
			return c.Status(400).SendString(err.Error())
		}

		return c.JSON(fiber.Map{
			"status": "success",
			"message": "Güvenli veri başarıyla işlendi",
		})
	})

	app.Listen(":8080")
}

Rust Tokio / Axum Handler

Yüksek performanslı Rust servisleri için GPG decryption katmanını Axum pipeline'ına dahil edin.

main.rs
use axum::{routing::post, Json, Router};
use aegis_tokio::DecryptLayer;
use serde::{Deserialize, Serialize};

#[derive(Deserialize)]
struct ProfileRequest {
    ssn: String,
    balance: f64,
}

#[derive(Serialize)]
struct ProfileResponse {
    status: String,
}

async fn handle_profile(Json(payload): Json) -> Json {
    // Şifresi otomatik çözülmüş payload'u güvenle işleyin
    Json(ProfileResponse {
        status: format!("Kimlik uzunluğu işlendi: {}", payload.ssn.len()),
    })
}

#[tokio::main]
async fn main() {
    let app = Router::new()
        .route("/api/secure-profile", post(handle_profile))
        .layer(DecryptLayer::new("/etc/aegis/server_private.key"));

    axum::Server::bind(&"0.0.0.0:8080".parse().unwrap())
        .serve(app.into_make_service())
        .await
        .unwrap();
}

Caddy Middleware Yapılandırması

Şifre çözme ve şifreleme işlemlerini doğrudan Caddy ters vekil (reverse proxy) katmanında gerçekleştirin. Herhangi bir dilde yazılmış arka ucunuzu anında Sıfır-Güven uyumlu hale getirin.

Caddyfile
{
    order aegis_http before reverse_proxy
}

:8080 {
    # Aegis işlemcisi içeren yönlendirme bloğu
    route {
        aegis_http {
            challenge_path /api/challenge
            login_path /api/login
            decrypt_requests
            encrypt_responses
            require_keyserver
            check_revocation
            tunneling_enabled
            server_email "api@example.com"
            server_passphrase "secret"
            server_private_key_path "/etc/caddy/server_private.asc"
            server_public_key_path "/etc/caddy/server_public.asc"
        }

        # Çözülmüş trafiği şeffaf şekilde standart backend'e iletir
        reverse_proxy localhost:3000
    }
}

Yerel Daemon Servisini Yükleyin ve Kaydedin (Native Messaging Host)

Native host, tarayıcı eklentisinin işletim sisteminizdeki GPG anahtarlığıyla güvenle konuşmasını sağlayan bir köprüdür. Platformunuza uygun yükleyiciyi seçin:

Windows: Resmi Releases sayfasından derlenmiş hazır aegis-native-host.exe kurulum dosyasını indirin.
macOS: Homebrew tap deposu üzerinden yükleyin:
Homebrew
brew install aegishttp/tap/aegis-native-host
Linux:
  • Ubuntu / Debian (APT PPA): Resmi PPA'dan yükleyin:
    Terminal
    sudo add-apt-repository ppa:aegis-http/stable
sudo apt update
sudo apt install aegis-native-host
  • Snap Store (Tüm dağıtımlar): Snap ile yükleyin:
    Terminal
    sudo snap install aegis-native-host
  • Arch Linux (AUR): AUR paketini yükleyin:
    Terminal
    yay -S aegis-native-host-bin
  • Kaynaktan Derleme:
    Terminal
    git clone https://github.com/AegisHttp/native-host-rust.git
cd native-host-rust
chmod +x install.sh
./install.sh
